# Crunchyroll Tampermonkey Skip Credit button removal Script
This is a simple button remover for the skip credit button, using tampermonkey 


# Step 1

get tampermoneky from the offical tampermonkey website https://www.tampermonkey.net/


# Step 2

Download the script and it should ask you if you want to install this script, click install and make sure tampermonkey is enabled, it should look like this if it works

Note: Refresh crunchyroll after installation
