# Crunchyroll Tampermonkey Skip Credit button removal Script
This is a simple button remover for the skip credit button, using tampermonkey 


# Step 1

get tampermoneky from the offical tampermonkey website https://www.tampermonkey.net/


# Step 2

Download the script and go to the tampermonkey extenstion's dashboard.


<img width="377" height="605" alt="image" src="https://github.com/user-attachments/assets/058d7544-c953-4a18-9352-df76343dd363" />


# Step 3

Go to utilites and import the file, or click the new script button and paste the code.
for more info read the tampermonkey FAQ https://www.tampermonkey.net/faq.php?locale=en&q=Q102
